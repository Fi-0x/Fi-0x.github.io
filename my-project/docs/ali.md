#### 奥德赛计划
- Current Path
: 若是按照你如今的路线持续下去5年，你的生活会是什么样？假想一天。
- Alternative Path
: 若是按照你另一道的路线持续下去5年，你的生活会是什么样？假想一天。
- Radical Path
: 如果你的经济状态不限和责任不限，你会怎么怎么做，5年后会怎么样？
