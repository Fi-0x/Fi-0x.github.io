嵌入式方向

- 单片机方向
    - MCU：STM32，N32，GD32、51
    - 内核：ARM，Cortex-M
    - 外设：定时器、串口、IIC、SPI等
    - 操作系统：实时操作系统FreeRTOS，ucos，RT-Thread
    - 语言：嵌入式C语言，ARM汇编
- Linux驱动工程师
    - MPU：i.MX6UlL,i.MX8,全志H3
    - 内核：ARM，Cortex-A
    - 外设：定时器，串口，IIC，MMU，HDMI，SPI等
    - 操作系统：Linux
    - Bootloader：U-Boot
    - 语言：嵌入式C语言，ARM汇编，脚本语言
- Linux应用工程师
    - 内核：ARM Cortex-A，Intel、AMD
    - 不涉及直接操作外设
    - 语言：C，CPP，PY，JAVA
- QT工程师
    - 跨平台
    - 不涉及直觉操作外设
    - 语言：C，CPP，PY
